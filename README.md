# ITfix

Professional IT Support website for Boca Raton, South Florida.

## Tech Stack

- **Framework:** Next.js 15 (App Router, Server Components)
- **Language:** TypeScript (strict)
- **Styling:** TailwindCSS
- **Backend:** Supabase
- **Testing:** Vitest + Playwright
- **i18n:** next-intl (en/ru)

## Development

```bash
npm install
npm run dev
```

## Project Structure

```
src/
├── app/           # Next.js App Router
├── components/    # Reusable UI components
├── lib/           # Utilities, Supabase client
└── styles/        # Global styles
```

## Phases

1. **Foundation** — Project setup, CI/CD, Docker
2. **Core** — Pages, components, Supabase integration
3. **Content** — SEO content, localization
4. **Launch** — Production deployment, monitoring
5. **Growth** — Features, optimization, analytics
