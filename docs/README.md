# ITfix Platform

Professional Technology Support for Homes and Small Businesses

## Mission

Provide reliable, professional onsite technology support for residential and small-business customers throughout Broward and Palm Beach County.

- Windows Installation & Repair
- Malware Removal
- Wi-Fi Troubleshooting
- Network Support
- Smart Home Installation
- Camera Installation
- Smart Locks & Doorbells
- Ethernet Cabling
- Low Voltage Infrastructure
